## objeto de estudio
El objeto de estudio es el desplazamiento corporal en el espacio a partir de datos de geolocalización.
## Interes personal 
Para este proyecto, mi intención no fue simplemente replicar un mapa convencional, sino explorar la posibilidad de “dibujar” mi recorrido por 
el Cajón del Maipo utilizando Processing a partir de los datos reales registrados con mi celular. Me interesaba que el propio trayecto se construyera de manera autónoma en pantalla, generándose en movimiento, como una especie de trazo vivo.
A su vez, quise incorporar los datos recopilados (como la velocidad) traduciéndolos visualmente a través del color, de modo que no solo se mostrara por dónde pasé, sino también cómo fue ese desplazamiento. Este enfoque toma como referencia 
la estética y lógica de visualización de Sensor Logger, buscando mantener una apariencia similar a sus mapas de localización, pero reinterpretada desde una perspectiva más experimental y gráfica.
En ese sentido, el proyecto no apunta únicamente a representar un recorrido, sino a construir una visualización dinámica donde el mapa se genera en tiempo real, integrando datos y movimiento como parte de una misma experiencia visual.
